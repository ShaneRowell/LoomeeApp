# Loomeé – Virtual Try-On & Measurement App

Loomeé is a 2D image-based virtual fitting application that allows users to upload a full-body image, visualize clothing overlays, and receive estimated body measurements.

## Features
- Full-body image upload
- 2D clothing overlay
- Pose detection and body segmentation
- Measurement estimation from images
- Size recommendation (S/M/L)
- Cloud-based storage and processing

## Tech Stack
- Frontend: Flutter
- Backend: Node.js + Express
- AI / CV: MediaPipe, TensorFlow Lite
- Database: MongoDB Atlas
- Storage: AWS S3 / Firebase Storage

## Repository Structure
- `frontend/` – Mobile application
- `backend/` – API and processing server
- `ai/` – Pose detection, measurements, overlay logic
- `cloud/` – Cloud and deployment configs
- `docs/` – Diagrams and reports
